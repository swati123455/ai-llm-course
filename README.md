# AI LLM Course – Build Real AI Applications

Welcome to the **AI LLM Course** repository.

This course teaches you how to build real-world AI applications using **Large Language Models (LLMs)**.

The course focuses on **practical implementation**, covering everything from AI fundamentals to deploying AI-powered applications.

---

# What You Will Learn

By following this course you will learn:

• How Large Language Models work
• Prompt engineering techniques
• Using LLM APIs
• Building AI applications with LangChain
• Creating Retrieval-Augmented Generation (RAG) systems
• Working with vector databases
• Fine-tuning models
• Deploying AI applications

---

# Projects In This Course

You will build several practical projects including:

1. AI Research Assistant (RAG system)
2. AI Code Assistant
3. Personal Knowledge Base AI
4. AI Automation Agent
5. Document Summarization Tool
6. Custom Fine-tuned AI Model

# Setup Instructions

1. Clone the repository

```
git clone https://github.com/swati123455/ai-llm-course.git
```

2. Navigate to the project

```
cd ai-llm-course
```

3. Create virtual environment

```
python -m venv venv
```

4. Activate environment

Mac/Linux

```
source venv/bin/activate
```

Windows

```
venv\Scripts\activate
```

5. Install dependencies

```
pip install -r requirements.txt
```
